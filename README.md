# RIIFO Indonesia Corporate Website

Multi-page corporate website for RIIFO Indonesia with interactive features.

## Features

- **Interactive Hero Banner**: Before/After image comparison slider
- **Product Systems Showcase**: Grid layout with product cards
- **Multi-page Structure**: Home, Company, Products, Contact, Warranty
- **Video Showcase**: Company page with embedded production video
- **Real Factory Photos**: Authentic RIIFO facility images

## Pages

| Page | Path | Description |
|------|------|-------------|
| Home | `/` | Company intro, product systems, before/after slider |
| Company | `/company` | Corporate info, global presence, video |
| Products | `/products` | Detailed product catalog |
| Contact | `/contact` | Inquiry form with Pardot integration |
| Warranty | `/warranty` | 25-year warranty certificate |

## Tech Stack

- Next.js 14 + TypeScript
- Tailwind CSS
- Vercel deployment
